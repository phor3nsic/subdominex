# NewFrogy
_Content Discovery_

### Single target:
```bash
./frogy.sh $x
```

### For multi targets:
```bash
for x in $(cat targets.txt);do ./frogy.sh $x;done
```

### Required Tools:

- subfinder
- assetfinder
- amass
- chaos
- sublist3r
- findomain
- subscraper
- shuffledns
- httpx
- gospider
- csprecon
